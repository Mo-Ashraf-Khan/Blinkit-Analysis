# Blinkit Sales Analysis Report
1. Project Overview

This project focuses on analyzing the sales performance, customer satisfaction, and inventory distribution of Blinkit. The objective is to derive meaningful insights using KPIs and interactive dashboards built in Power BI.

# 2. Business Requirement

The primary goal of this analysis is to:

Evaluate overall sales performance
Understand customer behavior through ratings
Analyze product distribution and outlet performance
Identify opportunities for business optimization
# 3. Key Performance Indicators (KPIs)
Total Sales
Total revenue generated from all items sold.
Average Sales
Average revenue per transaction.
Number of Items Sold
Total count of items sold across all outlets.
Average Rating
Average customer rating for products.

# 5. Charts & Analysis Requirements

# 5.2 Sales by Item Type
Objective: Identify top-performing product categories
Chart Type: Bar Chart
# 5.3 Fat Content by Outlet
Objective: Compare sales across outlets segmented by fat content
Chart Type: Stacked Column Chart
# 5.4 Sales by Outlet Establishment
Objective: Analyze impact of outlet age/type on sales
Chart Type: Line Chart
# 5.5 Sales by Outlet Size
Objective: Analyze correlation between outlet size and revenue
Chart Type: Pie / Donut Chart
# 5.6 Sales by Outlet Location
Objective: Understand geographic sales distribution
Chart Type: Funnel Map
# 5.7 All Metrics by Outlet Type
Objective: Provide a complete KPI comparison across outlet types

# 6. Key Insights (Example – You can customize after dashboard)
Low-fat products may contribute significantly to total sales
Certain item categories dominate revenue generation
Medium-sized outlets often generate higher sales
Urban locations tend to outperform rural outlets
Older outlets may show stable and consistent performance
# 7. Conclusion

This analysis helps in understanding Blinkit's sales patterns and customer preferences. The Power BI dashboard enables stakeholders to make data-driven decisions for improving sales, optimizing inventory, and enhancing customer satisfaction.
